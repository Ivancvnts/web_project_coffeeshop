# Triple Espresso

La pagina de Triple Espresso permite a los usuarios conocer mas acerca de la cafeteria, horarios de atención, recetar, acceso a reservación de mesas e información de contacto.

En este proyecto se utilizo tanto HTML como CSS, se aplico la metodologia BEM para nombrar y estructurar los archivos.

Planes de mejora del proyecto:

- Agregar un slider para los videos, de manera que si se agregan mas recetas los videos puedan deslizarse horizontalmente.
- Mejorar el estilo del campo de fecha para las reservaciones y hacer que tenga un estilo mas parecido al de la pagina.
